This project demonstrates CUDA-based image processing at scale. The software generates a batch of synthetic grayscale images and processes them on the GPU using two custom CUDA kernels: a 3x3 blur kernel and a Sobel edge detection kernel. The code supports command-line arguments, includes a Makefile and run script, and automatically writes proof-of-execution artifacts including sample output images and a timing log. The main lesson learned is that even simple image processing operations become meaningful CUDA workloads when they are applied to hundreds of inputs in a single execution.
